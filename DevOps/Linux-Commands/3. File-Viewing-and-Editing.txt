File Viewing and Editing
------------------------------
    1. cat <filename>: View the contents of a file.
    2. less <filename>: View file content one screen at a time.
    3. more <filename>: Similar to less, but with simpler functionality.
    4. head <filename>: View the first 10 lines of a file.
    5. tail <filename>: View the last 10 lines of a file.
    6. nano <filename>: Simple text editor.
    7. vim <filename>: Advanced text editor.
