User and Group Management
-------------------------
    1. useradd <username>: Add a new user.
    2. passwd <username>: Change a user's password.
    3. usermod -aG <group> <username>: Add a user to a group.
    4. groupadd <groupname>: Create a new group.
    5. deluser <username>: Delete a user.
