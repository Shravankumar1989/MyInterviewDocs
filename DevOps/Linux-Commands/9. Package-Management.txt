Package Management
------------------

1. For Debian/Ubuntu (APT-based)
---------------------------------
    1.1 apt update: Update package lists.
    1.2 apt upgrade: Upgrade installed packages.
    1.3 apt install <package>: Install a package.
    1.4 apt remove <package>: Remove a package.

2. For RedHat/CentOS (Yum-based)
--------------------------------
    2.1 yum update: Update installed packages.
    2.2 yum install <package>: Install a package.
    2.3 yum remove <package>: Remove a package.
