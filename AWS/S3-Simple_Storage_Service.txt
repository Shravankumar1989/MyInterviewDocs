1.  What is S3 (Simple Storage Service)?
----------------------------------------
    AWS that allows you to store and retrieve any amount of data, from anywhere, at any time.

2. What are Types of S3?
------------------------
    1.  S3 Standard: 
        - For frequently accessed data.
    
    2.  S3 Glacier: 
        - For archival data with longer retrieval times.
    
    3.  S3 Glacier Deep Archive: 
        - For long-term storage with the lowest cost and long retrieval times.
    
    4.  S3 Intelligent-Tiering: 
        - Automatically moves data between frequent and infrequent tiers based on usage.
    
    5.  S3 Standard-IA: 
        - For infrequently accessed data, but with immediate availability.
    
    6.  S3 One Zone-IA: 
        - For infrequent access, stored in only one availability zone.
    
    7.  S3 Outposts: 
        - For storing S3 data on-premises using AWS Outposts.
